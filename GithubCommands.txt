Creando ramas:

git branch ramaEjemplo   --creo
git checkout ramaEjemplo --me muevo a la rama
git add .				-- hago mi commit
git commit -m "todo a listas"
git push origin ramaEjemplo:RamaDiccionarios   --ramaEjemplo es mi rama el dos puntos: es para crear una rama en el Remoto


git log --oneline     ---para ver en que rama estoy


git push origin ramaEjemplo:RamaDiccionarios   --- actualizar la rama
